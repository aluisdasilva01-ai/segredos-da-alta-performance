# 3. Mentalidade de Crescimento — O Fundamento do Desenvolvimento

Baseado nos estudos da psicóloga Carol S. Dweck, essa é a base que define se você vai evoluir ou estagnar, independentemente do conhecimento técnico que tenha.

---

## 🧠 Dois tipos de mentalidade

### 🔒 Mentalidade Fixa
- Acredita que inteligência e talento são **dons naturais, imutáveis**
- Vê o esforço como sinal de pouca capacidade
- Evita desafios para não correr o risco de errar
- Interpreta o erro como **prova de incapacidade**
- Ignora críticas ou se sente ameaçado por elas
- Se incomoda com o sucesso de outras pessoas

### 🚀 Mentalidade de Crescimento
- Acredita que **toda habilidade pode ser desenvolvida com prática e método**
- Vê o esforço como **caminho para dominar algo**
- Busca desafios: é onde o aprendizado acontece
- Interpreta o erro como **informação para ajustar e evoluir**
- Usa críticas construtivas para melhorar
- Se inspira no sucesso de outras pessoas

---

## 📌 Por que isso importa?
Sem a mentalidade certa, nenhuma técnica ou ferramenta funciona. Com ela, qualquer limitação pode ser superada com tempo e dedicação.

---

## 💡 Aplicação para a área de Dados
- Se uma consulta SQL ou fórmula DAX não der certo → pense: *“Ainda não domino, mas vou estudar e ajustar”*
- Cada modelo de dados que falha é uma lição para fazer melhor na próxima vez
- Aceitar feedback sobre seus relatórios ajuda a entregar mais valor ao negócio
